Setting Up SSH Keys: A Magical Shortcut ü™Ñ

Before we dive into Git, let's add an enchanted touch with SSH keys. These keys work like magical passwords, granting you easy access to your GitHub kingdom. Here's how to conjure them:

  1. Open your Terminal or Git Bash
  2. Type: ssh-keygen -t rsa -b 4096 -C "your@email.com"
  3. Press Enter for default file location and an enchanting passphrase
  4. Your SSH key is born! Find it at ~/.ssh/id_rsa.pub
  5. In GitHub's kingdom, go to Settings → SSH and GPG keys → New SSH key
  6. Copy-paste your key and name it something magical
  7. Your keys are now synced between realms!

Linux Land üêß

  1. Open the Terminal - your wizard's wand!
  2. Type: sudo apt-get install git
  3. Wait for magic to install!
  4. Type: git config --global user.name "Your Name"
  5. Type: git config --global user.email "your@email.com"
  6. Create a repo spell: mkdir my_repo
  7. Enter: cd my_repo
  8. Craft a file potion: touch spellbook.txt
  9. Invoke Git: git init
  10. Add potion: git add spellbook.txt
  11. Cast spell: git commit -m "First spell cast!"
